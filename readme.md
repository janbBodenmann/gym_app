## Installation

Gym App Installieren:

```bash
  cd gymapp3
  cd gymapp3
  composer install
  cp .env.example .env
  php artisan key:generate

  npm install
  npm run build

  php artisan migrate

  php artisan serve --host=0.0.0.0 --port=8000
  http://10.5.43.8:8000/

  npx cap sync
  npx cap open android

```



Man muss sich immer im Ordner \GymApp3\gymApp3 befinden!
